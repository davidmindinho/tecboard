O que é Semântica?

A semântica em HTML refere-se ao significado e à estrutura do conteúdo na web. Utilizar tags semânticas ajuda a melhorar a acessibilidade, SEO e a compreensão do conteúdo por parte dos navegadores e leitores de tela.

Importância da Semântica

Usar tags semânticas corretamente permite que os motores de busca entendam melhor o conteúdo da página, o que pode melhorar o ranking nos resultados de pesquisa.
A tag <div> é uma tag genérica e não semântica, é usada para envolver elementos sem dar significado semântico ao conteúdo.
A tag <span> é uma tag inline genérica e não semântica, é usada para envolver elementos dentro de um parágrafo ou outro conteúdo inline.

Exemplos de Tags Semânticas

  • <header> - Define o cabeçalho de uma página ou seção.
  • <nav> - Define uma seção de navegação.
  • <main> - Define o conteúdo principal da página.
  • <article> - Define um artigo independente.
  • <section> - Define uma seção genérica de conteúdo.
  • <footer> - Define o rodapé de uma página ou seção.
  • <figure> - Define uma figura com legenda. <figcaption> é usado para adicionar uma legenda à figura.
